Quickstart (60 seconds)

Three steps to your first Watchdog Certificate:

# 1. Verify the API is alive
curl -s http://localhost:3400/healthz

# 2. See the catalog
curl -s http://localhost:3400/api

# 3. Buy a tier
curl -s -X POST http://localhost:3400/purchase/tier \
  -H "Content-Type: application/json" \
  -d '{"tier_id": "pro", "customer_email": "you@yourco.com"}'

Auth (Day 1+)

Currently the API is open. In production (Day 7+, after first £199/mo charge), all POSTs will require an X-API-Key header.

curl -s http://localhost:3400/api/whoami \
  -H "X-API-Key: sk_live_..."

Stripe webhook setup

To enable real Stripe payments:

  1. Set STRIPE_SECRET_KEY + STRIPE_WEBHOOK_SECRET in csoai-mcp-monetization/.env
  2. In Stripe dashboard, add webhook endpoint: https://your-host/webhook/stripe
  3. Subscribe to events: checkout.session.completed, customer.subscription.created, invoice.paid, payment_intent.succeeded
  4. Use POST /webhook/test for testing without a real payment
