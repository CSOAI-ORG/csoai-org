## 🚀 Quick Start

```bash
# TypeScript SDK
npm install meok-sdk-ts@3.1.0

# Python SDK (when PyPI clears)
pip install meok-sdk==3.1.0

# CLI
meok csoai status
meok csoai map
meok csoai dome
```

## ✨ v3.1 Features

- **14 Static Pages** — Landing, Map, Crosswalk, DOME, Council, Sigil, Comply, Charter, Training, Ecosystem, Prosperity
- **6-Region Compliance Map** — EU, US, UK, China, Singapore, Korea with live countdowns
- **52-Framework Crosswalk** — EU AI Act, NIST, ISO 42001, DORA, NIS2, CRA, FDA, HIPAA, SOC 2, FedRAMP + 42 more
- **BFT Council** — Byzantine Fault Tolerant governance with voting UI
- **SIGIL Verification** — Ed25519 certificate attestation with blockchain anchoring
- **A2A Agent Card** — Google A2A v1.0 compliant agent descriptor
- **4 Cloud Run MCPs** — eu-ai-act, dora, nis2, cra (all LIVE)
- **x402 Monetization** — Per-call USDC billing on Base
